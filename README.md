# 智子神经节点 (Agentic Neural Nodes)

本目录为系统底层 AI Agent 的核心定义与 Prompt 控制中枢。每一个 `.md` 文件代表一个独立挂载的人格模型或特定领域的处理管线 (Pipeline)。设计遵循“高信噪比、极简、精准”的战略架构师准则。

## 🧠 分拣目录架构 (Agent Taxonomy)

### 🔬 学术与研究管线 (Academic & Research)
- `paper-analyst.md`: 论文深度拆解与分析引擎。
- `paper-reviewer.md`: 模拟顶会审稿人视角的严格论文审查。
- `xray-paper.md`: “X光”透视扫描器，精准提取论文骨架与核心创新点。
- `scientific-illustrator.md`: 科学配图与视觉化逻辑结构表达构思。
- `translate-paper-from-chs2eng.md`: 学术级中文到英文精密翻译管线。

### 💼 商业与领域专家 (Business & Domain Experts)
- `industry-analyst.md`: 行业研报生成与商业洞察拆解。
- `cn-emr-l5-expert.md`: 中国电子病历 (CN-EMR) L5 级架构专家，医疗信息化垂直领域专属。

### ⚙️ 认知与思想淬炼 (Cognitive & Thought Processing)
- `cognitive-engine.md`: 高阶认知引擎，打破思维定势与逻辑盲区。
- `cognitive-pipeline-architect.md`: 认知管线架构师，构建与规划复杂系统思考流。
- `concept-analyzer.md`: 概念解析器，从本质上对复杂概念进行“物理级”降维拆解。
- `core-concepts-100.md`: 百大核心元概念与心智模型调用映射。
- `grounded-insights.md`: 扎根理论洞察，从繁杂非结构化数据中抽取底层规律。
- `thought-refiner.md`: 思想提纯器，极速提升文本信息熵，淘汰冗余。

### 📖 深度理解与解构 (Deep Comprehension & Reconstruction)
- `Youtube-Summary.md`: 视频逻辑逆向工程与深度叙事重构管线，剥离废话提取核心变量。
- `book-companion.md`: 私人读书伴读，深度对话式知识摄取。
- `xray-book.md`: 书籍解构器，快速透视原著知识树与论点架构。
- `reading-insight.md`: 专注深层次阅读洞察提取。

### ✍️ 创作与表达引擎 (Creation & Expression)
- `writing-assistant.md`: 全能型写作辅助与散文体渲染强化。
- `blogger-formatter.md`: 针对博客/平台分发的阅读体验与视觉排版优化。
- `paper-editing.md`: 论文润色与学术术语格式化。
- `Translation.md`: 通用高精度中英等语种翻译器。

### 🧭 探索与辅助 (Exploration & Companionship)
- `human-nature-research.md`: 人性与行为模型探测，归因底层群体动机。
- `soul-companion.md`: 灵魂挂载模式，提供深度精神共情与心理投射解析。

## 🚀 部署与运行时协议 (Deploy Protocols)

1. **精准挂载 (Precision Mount)**：面对业务需求，按需挂载匹配的 Agent 实例，杜绝能力错配。
2. **约束铁律 (Negative Constraints)**：严格遵循各配置文件内的负面约束（如：禁止无意义寒暄、禁止过度列点），绝不允许系统妥协输出标准。
3. **高频突变 (Mutation Cycle)**：Agent 的迭代更新需遵循 `forge` 模式的测试基准，保持演化优势。
